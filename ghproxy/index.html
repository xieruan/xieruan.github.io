GitHub Proxy

GitHub 文件 , Releases , archive , gist , raw.githubusercontent.com 文件代理加速下载服务.

使用说明

本站使用 香港AWS月抛 线路,适用于各种网络,如体验不佳,可尝试访问其余加速站使用.

终端命令行

支持终端命令行 git clone , wget , curl 等工具下载.
支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.
注意：不支持 SSH Key 方式 git clone 下载.

git clone

git clone http://xrproxy.top/https://github.com/stilleshan/ServerStatus

wget & curl

wget http://xrproxy.top/https://github.com/stilleshan/ServerStatus/archive/master.zip
wget http://xrproxy.top/https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile
curl -O http://xrproxy.top/https://github.com/stilleshan/ServerStatus/archive/master.zip
curl -O http://xrproxy.top/https://raw.githubusercontent.com/stilleshan/ServerStatus/master/Dockerfile

首页下载

在本页地址栏输入合规链接（参考以下链接）点击下载按钮
支持 raw.githubusercontent.com , gist.github.com , gist.githubusercontent.com 文件下载.
